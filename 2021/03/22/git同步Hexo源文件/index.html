git同步Hexo源文件

本文最后更新于：2021年9月20日 晚上

Hexo上传到github的是生成的静态网页，不包含源文件，可以另开一个仓库用于储存Hexo的源文件。

使用Git*托管

新建branch

我放在Gitee的私有仓库，国内访问速度和延时好一些。

新建仓库或者给Blog仓库创建分支都可以，我给原Blog仓库创建了一个名为hexo的分支，然后把它设为主分支。

clone到本地

在本地Blog文件夹下git clone 仓库地址
